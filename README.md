## Install

```shell
npm install evsmc-base-ui -S
```

## Quick Start

``` javascript
import Vue from 'vue'
import Element from 'element-ui'
import EvsmcBaseUi from 'evsmc-base-ui'

Vue.use(Element)
Vue.use(EvsmcBaseUi)

// or
import {
  Add,
  BatchButton
  // ...
} from 'evsmc-base-ui'

Vue.use(Add)
Vue.use(BatchButton)
```

For more information, please wait our documentation.

## Browser Support

Modern browsers and Internet Explorer 10+.
